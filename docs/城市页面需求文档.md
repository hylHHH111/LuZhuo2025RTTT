# 鹭卓2025 RTTT巡演记录网站 - 城市页面需求文档

## 项目概述

为成都、合肥、太原、北京、南京五个城市站页面实现"卡册式"逐层展开交互风格，展示现场物料，保留核心模块，保证视觉统一、交互流畅、多终端适配。

---

## 一、核心需求

### 1.1 页面结构

```
┌─────────────────────────────────────────────────────────────────┐
│                        顶部区域 (Header)                         │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  城市站名称 + 主标题 + 全局导航 (与home.html一致)         │   │
│  └─────────────────────────────────────────────────────────┘   │
├─────────────────────────────────────────────────────────────────┤
│                       上半区三栏布局 (60%:20%:20%)               │
│  ┌──────────────────────────┬──────────┬──────────┐           │
│  │                          │          │          │           │
│  │      纪录片模块           │ 文字动态 │ 垂直导航 │           │
│  │      (60%)               │  (20%)   │  (20%)   │           │
│  │                          │          │          │           │
│  └──────────────────────────┴──────────┴──────────┘           │
├─────────────────────────────────────────────────────────────────┤
│                    间距: 32px-40px                              │
├─────────────────────────────────────────────────────────────────┤
│                       下半区三栏布局 (1:1:1)                     │
│  ┌─────────────────┬─────────────────┬─────────────────┐      │
│  │                 │                 │                 │      │
│  │   海报花絮区     │    直拍区        │    图集区       │      │
│  │   (单层/弹窗)    │  (三层级卡册)    │  (三层级卡册)   │      │
│  │                 │                 │                 │      │
│  └─────────────────┴─────────────────┴─────────────────┘      │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 卡册交互规范

| 区域 | 卡片比例 | 标准尺寸 | 交互层级 | 交互类型 |
|------|----------|----------|----------|----------|
| 纪录片 | 16:9 | 自适应 | 单层 | 视频播放 |
| 文字动态 | 自适应 | 自适应 | 单层 | 滚动展示 |
| 垂直导航 | 固定按钮 | 自适应 | 单层 | 弹窗展示 |
| 海报花絮 | 3:4 | 584×780 | 单层 | 弹窗卡册 |
| 直拍区 | 3:4 | 584×780 | 三层级 | 场次→舞台→视频 |
| 图集区 | 3:4 | 584×780 | 三层级 | 类别→造型→图片 |

**三层级卡册交互流程：**
```
层级1: 顶层卡片 (Day1/Day2/定妆照等)
    ↓ 点击展开
层级2: 分类选择 (舞台/造型)
    ↓ 点击展开  
层级3: 内容展示 (视频/图片集)
```

### 1.3 模块功能需求

#### 1.3.1 纪录片模块
- **功能**: 视频播放、全屏、进度控制
- **交互**: 
  - 播放/暂停按钮
  - 进度条拖动
  - 全屏切换
  - 音量控制
- **异常处理**: 
  - 加载动画
  - 播放失败提示
  - 刷新按钮
- **视觉**: 自定义播放器样式，与主题色一致

#### 1.3.2 文字动态区
- **功能**: 公告/文字内容滚动展示
- **交互**:
  - 自动上下滚动
  - Hover时暂停滚动
  - 滚动速度可调
- **装饰**: 简约手绘线条边框
- **内容**: 支持HTML格式文本

#### 1.3.3 垂直导航区
- **导航项**: 合照区、花束区、官摄区 (共3项)
- **交互**: 
  - 点击弹出卡册弹窗
  - 弹窗内网格布局展示图片
  - 支持图片大图查看
  - 支持轮播浏览
- **异常处理**: 图片加载失败占位图

#### 1.3.4 海报花絮区
- **布局**: 顶层封面卡片 (3:4比例)
- **交互**:
  - 卡片中央圆形点击按钮
  - 点击弹出图片卡册弹窗
  - 弹窗内网格展示海报/花絮图片
  - 支持大图查看、轮播
- **视觉**: 卡片悬停效果、阴影

#### 1.3.5 直拍区
- **顶层卡片**: Day1、Day2 (3:4比例)
- **三层级交互**:
  ```
  层级1: Day1 / Day2 卡片
      ↓
  层级2: 舞台选择 (主舞台/延伸台/互动区等)
      ↓
  层级3: 视频列表 (各舞台直拍视频)
  ```
- **功能**: 
  - 视频卡片展示
  - 视频播放功能
  - 弹窗内导航返回
- **异常处理**: 视频加载失败提示

#### 1.3.6 图集区
- **顶层卡片**: 定妆照、Day1、Day2 (3:4比例)
- **三层级交互**:
  ```
  层级1: 定妆照 / Day1 / Day2 卡片
      ↓
  层级2: 服装造型选择
      ↓
  层级3: 图片集 (支持下载)
  ```
- **功能**:
  - 图片大图查看
  - 轮播浏览
  - 可选下载功能
- **弹窗**: 统一弹窗样式，支持层级导航

## 三、布局规范

### 3.1 上半区布局 (60%:20%:20%)

```css
.upper-section {
    display: grid;
    grid-template-columns: 3fr 1fr 1fr;  /* 60%:20%:20% */
    gap: 24px;
    margin-bottom: 32px-40px;
}

/* 响应式: 平板端 */
@media (max-width: 1024px) {
    .upper-section {
        grid-template-columns: 2fr 1fr;  /* 纪录片占2列，其他堆叠 */
    }
}

/* 响应式: 移动端 */
@media (max-width: 768px) {
    .upper-section {
        grid-template-columns: 1fr;  /* 全部堆叠 */
    }
}
```

### 3.2 下半区布局 (1:1:1)

```css
.lower-section {
    display: grid;
    grid-template-columns: repeat(3, 1fr);  /* 1:1:1 */
    gap: 24px;
}

/* 响应式: 平板端 */
@media (max-width: 1024px) {
    .lower-section {
        grid-template-columns: repeat(2, 1fr);  /* 2列 */
    }
}

/* 响应式: 移动端 */
@media (max-width: 768px) {
    .lower-section {
        grid-template-columns: 1fr;  /* 单列 */
    }
}
```

### 3.3 间距规范

| 区域 | 间距值 | 说明 |
|------|--------|------|
| 上下半区之间 | 32px-40px | 固定间距 |
| 模块内部 | 16px-24px | 元素间距 |
| 卡片间距 | 24px | grid gap |
| 页面边距 | 24px (桌面) / 16px (移动) | 响应式 |
| 内容最大宽度 | 1400px | 居中限制 |

## 四、交互规范

### 4.1 卡册展开动画

```css
/* 卡片悬停效果 */
.card {
    transition: all 0.3s ease;
    transform-style: preserve-3d;
}

.card:hover {
    transform: translateY(-8px) scale(1.02);
    box-shadow: 0 12px 32px rgba(91, 164, 201, 0.25);
}

/* 弹窗动画 */
.modal {
    animation: modalIn 0.3s ease;
}

@keyframes modalIn {
    from {
        opacity: 0;
        transform: scale(0.9) translateY(20px);
    }
    to {
        opacity: 1;
        transform: scale(1) translateY(0);
    }
}
```

### 4.2 层级导航

```
┌─────────────────────────────────────────┐
│  ← 返回上级    当前层级标题      [X]关闭 │  ← 导航栏
├─────────────────────────────────────────┤
│                                         │
│           内容展示区域                    │
│         (卡片网格/视频/图片)              │
│                                         │
└─────────────────────────────────────────┘
```

- **返回按钮**: 左侧箭头图标，点击返回上一级
- **标题**: 居中显示当前层级名称
- **关闭按钮**: 右上角，点击关闭整个弹窗

### 4.3 响应式断点

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| 桌面端 | > 1024px | 完整三栏布局 |
| 平板端 | 768px - 1024px | 两栏或混合布局 |
| 移动端 | < 768px | 单栏堆叠布局 |
| 小屏移动 | < 480px | 紧凑布局，卡片缩放 |

- **基础**: HTML5 + CSS3 + JavaScript (ES6+)

## 八、注意事项

1. **视觉统一**: 所有页面必须与home.html配色、字体保持一致
2. **交互流畅**: 动画时长控制在0.3s以内，确保响应速度≤0.5秒
3. **图片比例**: 所有卡片严格保持3:4比例，等比缩放
4. **异常处理**: 每个模块都要考虑加载失败、网络异常等情况
5. **代码复用**: 通用组件抽离，避免重复代码
6. **注释规范**: 关键逻辑添加中文注释
